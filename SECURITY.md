## Vulnerability Reporting

**DO NOT CREATE PUBLIC ISSUES** for security vulnerabilities. Instead:

1. **Encrypted Disclosure**: Email security@yourdomain.com with:
   - Subject: `[SECURITY] Vulnerability in Trading-AI`
   - Body encrypted using our [PGP Key](#pgp-public-key)
   - Include:
     - Vulnerability description
     - Reproduction steps
     - Impact assessment

2. **Response SLA**:
   - Acknowledgement: Within 24 hours
   - Patch timeline: Critical fixes within 72 hours
   - Public disclosure: Coordinated release after patch
